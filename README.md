# API-PostreSQL

School work of Data Base II

---
### Install dependecies:
```
$ npm install
```
### Create ```.env``` file with the following info:
```
POSTGRES_USER=yourUserName
HOSTNAME=localhost
POSTGRES_DATABASE=Library
POSTGRES_PASWORD=yourPassword
POSTGRES_PORT=yourPort
```
### run project with:
```
$ npm start
```
